# SketchUI Library

A simple UI library for Processing with Button, Slider, and TextField components.

### Slider

### TextField

### Button Bar
- Create button bar with.
- Register button action function.
- Call to draw it.
- Run the action button when mouse pressed.


## Installation

1. Download the  folder.
2. Move it to your  directory in the Processing sketchbook.
3. Restart Processing, and it will appear in the "Sketch > Import Library" menu.
